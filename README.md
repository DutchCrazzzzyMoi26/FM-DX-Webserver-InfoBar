# DutchCrazzzzyMoi26 FM-DX InfoBar

A customizable information bar for FM-DX Webserver.

This info bar can show RSS feed items, custom text messages and rotating information lines directly on your FM-DX Webserver page.
It is useful for showing news headlines, or other live information.

---


![DutchCrazzzzyMoi26 FM-DX Info Bar preview](https://www.pd2pw.nl/Github/screenshot.png)


---


## Requirements

To use this info bar you need:

- FM-DX Webserver
- Access to your FM-DX Webserver files


Optional, if you use the RSS updater script:

- Python 3
- The required Python modules used by the script
- Permission to write/update the text file used by the info bar
